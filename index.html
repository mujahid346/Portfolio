Student Placement Prediction Model

  • Built a neural network using TensorFlow/Keras to predict whether a student will be placed based on academic and training data.
  • Preprocessed data using Pandas and Scikit-learn, including feature scaling, label encoding, and train-test splitting.
  • Achieved an accuracy of 81% using a 3-layer neural network with ReLU and sigmoid activation functions.
  • Evaluated model performance using confusion matrix and accuracy score.
  • Tools: Python, TensorFlow, Keras, Pandas, Scikit-learn.
